# MCQ Generator using Python

## Overview
This repository contains a Python-based MCQ (Multiple Choice Question) generator
developed as a learning project to understand Python packaging, modular code design,
and basic text processing techniques.

## Motivation
The goal of this project was to explore:
- How to structure a Python project using a `src/` layout
- Creating reusable modules
- Managing dependencies using `requirements.txt`
- Packaging a Python application using `setup.py`

## Project Structure
src/ # Core source code
experiment/ # Experimental and testing scripts
setup.py # Package configuration
requirements.txt # Project dependencies
